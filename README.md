==================================

### Setup
* Install `erlang-esdl` and `erlang-esdl-dev`.
* Compile with `erlc maze_gen.erl ui.erl`

### Run
`erl -pa /usr/lib/esdl/ebin/ -run ui demo`
Hit `Esc` to close the SDL window.
